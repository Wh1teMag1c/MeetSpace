# MeetSpace — Сервис бронирования переговорных комнат

Веб-приложение для автоматизации управления офисными пространствами.

## Стек технологий
- **Backend:** Django, DRF
- **Database:** PostgreSQL
- **Frontend:** React (Vite), Bootstrap 5
- **Infrastructure:** Docker, Nginx
